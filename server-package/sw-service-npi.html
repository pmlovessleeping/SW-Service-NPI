樣品與硬體準備原則

請依申請項目準備對應樣品與硬體條件，避免因樣品或配件不足延誤開發與驗證。

SUSI Porting 樣品準備原則
  • 提供標準可開機整機，須包含 CPU、RAM、硬碟、散熱片/散熱器與 Adapter，並可正常進入作業系統。
  • 如有外接模組且與 SUSI 功能相關，例如外接的 GPIO 擴充卡，則必需提供外接模組。
OS Image Porting 送測機器與硬體準備規範
1. 基礎出貨機器
  • 必須是標準可開機整機。
  • 需包含 CPU、RAM、硬碟、散熱片/散熱器、Adapter。
  • 若 PSU 有特殊高規格需求，須一併附上。
2. AMT 功能需求
  • 若有 AMT 功能，請提供有 AMT 功能的機器。 若機器可正常長出 AMT COM，也可作為 AMT 判定依據。
  • 若 PCH 分不同 SKU，須提供有 AMT 的 PCH SKU。
3. 模組、擴充板與架構
  • OS Image 若搭配擴充板，須提供擴充板。
  • 採上下板或主板+載板架構者，上下板皆須提供。
  • 若要安裝 WiFi/BT/NPU/外接顯卡等 Driver，且須提供對應 Module。
4. 線材與介面
  • 特殊車用或圓形連接器 I/O 必須提供線材。
  • 如有非普通接頭的 Internal USB 則必須提供。
  • COM Port 線材若屬 Optional 但預設有長出，也必須提供。
5. I/O 與功能 SKU 選擇
  • 若同一型號存在不同 SKU，且各 SKU 的 LAN / COM Port 數量不同，請優先提供一台包含完整 I/O 配置的機器；若無完整 I/O SKU，則須提供所有差異 SKU 的機器。
  • 若 COM/CANBus 共用 Port 優先提供 COM 機器。
  • Audio 轉板雖屬 Optional 仍須提供。例外情況：不裝 Driver、走 inbox，且 QA 不測 Audio 時，可不提供 Audio 轉板。
